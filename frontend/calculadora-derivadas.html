Clasificación

¿Cómo llegamos a estos resultados?

  1. Calculamos la primera derivada de la función cuadrática: f'(x) = 2ax + b.
  2. Igualamos la derivada a cero para encontrar el punto crítico: 2ax + b = 0.
  3. Despejamos x* y evaluamos la función original para obtener (x*, f(x*)).
  4. Revisamos el signo de la segunda derivada f''(x) = 2a para clasificar el punto como mínimo o máximo.
